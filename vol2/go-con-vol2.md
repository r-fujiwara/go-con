gdb
===

### gdb 教えて下さい！(1/2)
```
$ git clone git@github.com:r-fujiwara/hiromi-go.git
$ cd path/to/hiromi-go.git
$ go build -gcflags "-N -l" hiromi.go
$ gdb hiromi
```
